# Weather Data Pipeline - India

## Project Overview

This project collects real-time weather data from 35+ cities across India using the OpenWeather API.

The collected data is stored in CSV format and analyzed using Python, Pandas, Matplotlib, and Seaborn.

## Features

* Real-time weather data collection
* Multi-city support (35+ cities)
* Automatic CSV storage
* Data cleaning
* Exploratory Data Analysis (EDA)
* Weather trend analysis
* Correlation analysis

## Tech Stack

* Python
* Pandas
* Requests
* Matplotlib
* Seaborn
* OpenWeather API
